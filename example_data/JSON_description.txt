group_name * - строка, название группы
days[] - массив дней недели

День недели
weekday * - число, номер дня недели (нумерация начинается с 1 - понедельник, заканчивается 7 - воскресенье)
lessons[] - массив занятий

Занятие
subject * - строка, название предмета
type * - число, тип занятия (лекция, лабораторная работа, практика (справка по типам ниже))
time_number * - число, порядковый номер пары в дне (1-пара в 9:00, 2 - в 10:35, 4 - в 14:00, 7 - в 18:45)

teachers[] - массив преподавателей. если нет данных, указывается null, либо пустой массив []
teacher_name - строка, имя преподавателя

auditories[] - массив аудитории, если нет данных, об аудитории, указывается null, либо пустой массив []
auditory_name - строка, название аудитории

parity - число, периодичность занятия (0 - еженедельно, 1 - по нечетным неделям, 2 - по четным неделям)

Примечания: 
Все параметры обязательны.
Параметры, помеченные звездочкой ( *) не могут быть null

Типы занятий
0 - практическое занятие (если нет типа, по-умолчанию)
1 - лабораторная работа
2 - лекция
3 - семинар
4 - консультация
5 - внеучебное занятие
6 - зачет
7 - экзамен
